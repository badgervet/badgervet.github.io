A.A.S. in Software Development

The A.A.S. in CIS – Software Development is an entry-level program focused on building practical software development skills. Students learn programming in Python, C#, C++, and Java, along with full-stack web development (PHP, MySQL, Bootstrap), object-oriented design, and real-world project planning, while earning the CIS-Applied Systems & Design Certificate and developing a portfolio of functional applications.

A.A.S. in Game Design

The A.A.S. in CIS – Game Design is an entry-level program designed for students interested in developing video games and interactive applications. The curriculum covers programming in Python and C#, game engine development, full-stack web technologies, database design, and comprehensive software project planning, culminating in the CIS – Game Design Certificate, CIS-Applied Systems & Design Certificate, and a professional portfolio.

A.A.S. in Information Technology Generalist

The A.A.S. in CIS – Information Technology Generalist is an entry-level program that provides a broad foundation in information technology. Students gain essential skills in programming (Python and C#), databases, web development, networking fundamentals, cloud computing, and security, while earning the CIS-Software Foundations and CIS-Applied Systems & Design Certificates to prepare for a variety of entry-level IT positions.

A.A.S. in Networking/Cybersecurity

The A.A.S. in Computer Networking/Cybersecurity is an entry-level program that prepares students for careers in network administration and cybersecurity. Students build foundational skills through CompTIA A+, Network+, Security+, CCNA, Server+, Linux+, cloud fundamentals (AZ-900/SC-900), PenTest+, and CySA+ certifications while earning three stackable certificates, including the CIS-IT Foundations Certificate and culminating with the CIS-Cybersecurity Technician Certificate.

Core Technologies

CompTIA Certifications CCNA Internships
